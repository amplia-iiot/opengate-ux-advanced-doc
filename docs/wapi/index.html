OpenGate-UX API to allow publish and manage all external widgets

Error codes ¶

Code Message
4000 Workspace ${workspace_id} not found
4001 Workspace cannot be deleted.
4100 User domain is not equals to domain in the request
4101 User profile not allowed
4102 User not found
4103 Session not found
4104 Session expired
4105 Widget not found
4106 Widget already exist
4107 File meta is not attached on request
4108 Meta file invalid. Widget actions not found into widget definition. Widget actions not found <$widgets_error>, widget loaded <$widget_loaded>
4109 File vendor is not attached on request
4110 File bundle is not attached on request
5000 Internal server error. There are bbdd problems. Please consult with your administrator.

WiWi ¶

This collection of endpoints allow you publish/modify/delete/read all widgets that you have accessible into your domain. Each widget that you publish only could be visible into yours domain. Only a user with root, admin_domain or admin profiles could publish/modify/delete a widget.
